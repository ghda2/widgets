Widget de Chat - Demonstração

Este é um exemplo de como integrar o widget de chat em seu site. O widget aparece como um botão flutuante no canto inferior direito.

Como usar:

Adicione o seguinte código ao final da tag <body> do seu HTML:

<script src="http://localhost:8001/widget.js"></script>

Para produção, use o domínio do seu servidor:

<script src="https://bot.nexr.me/widget.js"></script>

Funcionalidades:

  • Botão flutuante para abrir/fechar o chat
  • Interface responsiva
  • Envio de mensagens para o servidor
  • Feedback visual durante o envio
  • Suporte a teclado (Enter para enviar, ESC para fechar)

Teste o Widget:

Clique no botão azul no canto inferior direito para abrir o chat e testar o envio de mensagens.
